This website is a 2D light playground. The ligthing algorithm is called Holographic Radiance Cascades. More technical details are on the github page. Source Code
Here you can switch between different modes, these are in development with more to be added.
Upload a video to start the light show.
For the lyrics mode upload an audio file with this naming convention for it to correctly work: songname-artist.mp3
This is the quality of the lighting. (Resolution of the HRC grid). Higher means more quality but lower performance and vice versa.
